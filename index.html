Github guide

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere. In this tutorial, i will teach you how to make a website and how to add some basic stuff. To complete this tutorial, you need a GitHub account, basic coding knowledge and Internet access. To get started, make a repository like shown on the image down below, call it "yourusename".github.io-test" and make a new file called "index.html" and one called "style.css". Enter some random words in "index.html" and commit to master (your main branch). While we wait for the website to upload, go to to settings (the gear in the about page) and fill out the "website" part with the suggested one.

Your website is ready when it shows a green checkmark next to the "commit" button. Go to the link (the one you typed in the setting thing) and, if everything is working correctly, you should be able to see your webpage displaying correctly, with the word you typed in before. Now that everything works, we are now ready to code!

Go in the "index.html" file and write first <!DOCTYPE html>, to tell the app which code language are you using: yes, Github supports all type of coding languages. Then write <html> and in a new line </html>. Then, add <head> and </head> in between the two <html>'s. In between head, add this line of code: <link rel="stylesheet" href="style.css">. This line will link the "stlye.css" file that we made before with the index.html: this will be used to style your page. Down below you find some of the basic css and html tags, that you will use to make your personal webpage, and also some resources that you can use to learn more about the topic

Html guide

HTML stands for HyperText Markup Language. It is a standard markup language for web page creation. It allows the creation and structure of sections, paragraphs, and links using HTML elements (the building blocks of a web page) such as tags and attributes.

<p>

<p> is a paragraph. This is where you will write most of your basic text in.

<ul> <li>

"ul" is a list, and "li" is the bullet point. In case you want a numbered one, use "ol" in place of "ul".

<h1> - <h6>

The h's are headings. They range in size from <h1> and <h6> (the smallest)

<img> <vid>

Used to add images and videos to your webpage

<body> <head>

The body contains meta information about the page, while the body is where you will write everything that will be displained in the webpage

<div>

Used to divide the webpage and it is fundmental for css (in this page there are more than 50)

<header>

The top part of the webpage. In this case i use it as an index.

<code>

In case you know some Javascript, used to add some code to the webpage

Css guide

CSS stands for Cascading Style Sheets: CSS describes how HTML elements are to be displayed on screen, paper, or in other media, and it saves a lot of work. It can also control the layout of multiple web pages all at once

color

The color of something. Works for text and elements. It works with hex codes.

background-color

The color of an item. Also uses HEX codes

text-align

Where the font should allign in the webpage. Inputs are center,

margin

The margin of any item or box. Can be specifed with margin-left, down and so on.

font-size

The size of any text. Uses px (pixels) as unit.

display:grid

Uses a grid to make elements look nice. This whole webpage wouldn't be possible without this.

font-family

The actual font of the text. Examples are monospace, arial or gothic.

display:flex

Works as a support for everything else in the webpage
